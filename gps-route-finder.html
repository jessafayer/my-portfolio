GPS Route Finder

Project Overview

This project demonstrates the implementation of a GPS system to find the shortest route between cities using the Breadth-First Search (BFS) algorithm. The system is built using Python and represents cities as nodes in a graph, with edges representing the routes between them.

View Objective

Simulation Code

The GPS route finder is implemented using a class-based structure, with classes representing cities and the map. The BFS algorithm is used to determine the shortest path between two cities.

Visualization

The visualization script uses the networkx library to plot the graph of cities and routes, showing the shortest path determined by the BFS algorithm.
