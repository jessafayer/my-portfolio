Monty Hall Simulation

Project Overview

This project simulates the famous Monty Hall Problem, demonstrating the Central Limit Theorem through multiple iterations and conditions. The simulation shows how the probability of winning changes when switching doors versus not switching doors.

View Objective

Simulation Code

The simulation is implemented in Python, using a class-based approach to encapsulate the Monty Hall game logic.

View Simulation Code

Visualization

The visualization script uses Seaborn to plot the results of the simulation, showing how the win percentages approach the expected probabilities over multiple iterations.

View Visualization Code

Results and Analysis

The simulation results align with the theoretical probabilities, demonstrating that switching doors yields a higher win percentage over many iterations.
